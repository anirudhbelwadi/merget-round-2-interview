### Frontend Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

The frontend will run on `http://localhost:3000` and automatically open in your browser.

### Usage

1. Click on any prompt in the tree to view its details
2. Use the "Notes" tab in the side panel to add and view notes
3. Expand/collapse prompts to see their nodes
